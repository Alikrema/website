
## A More Complex Function

Let's look at a function that adds two numbers:

```javascript
function add(a, b) {
  // This is a simple addition function
  return a + b;
}

const result = add(5, 3);
console.log(`The result is: ${result}`); // Expected output: The result is: 8
```
